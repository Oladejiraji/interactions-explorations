# Pip-Boy UI — Feature Ideas

## Interactive Features

- **Editable stats** — click a stat bar to increment/decrement values, with sound feedback
- **Drag-and-drop inventory** — reorder items or move between categories
- **Working radio tab** — simple audio player with static between "stations" (procedural tones/melodies)
- **Map tab** — stylized grid map with blinking location markers (canvas or SVG)
- **Boot sequence** — animated startup screen on page load (text lines appearing one by one, terminal-style)

## Visual Polish

- **Vault Boy silhouette** — SVG figure on the STAT tab that highlights limbs based on stats
- **Screen glitch on hover** — random horizontal offset/color shift when hovering the bezel edge
- **Amber/white/green color modes** — toggle between classic Pip-Boy screen colors (like Fallout 4 settings)
- **CRT power on/off** — shrinking dot animation when navigating away, expanding when arriving

## Data & Interaction

- **Quest log (DATA tab)** — list of quests with completion checkboxes
- **Perks system** — tree/grid of unlockable perks based on stat values
- **Weight tracking** — total carry weight vs max, with "over-encumbered" warning
- **Working clock** — real-time clock displayed in Pip-Boy style in the header

## Fun Extras

- **Keyboard navigation** — arrow keys to browse, tab to switch sections (like the actual game)
- **Easter eggs** — hidden items in inventory, Konami code unlocking something
